TASK 1 Prove Shallow Clone Bug
▼
  • Create const orig = { name: "Priya", addr: { city: "Jaipur" } }.
  • Make shallow copy with spread: const copy = { ...orig }.
  • Mutate copy.addr.city = "Mumbai" and log orig.addr.city.
  • Use structuredClone(orig) and repeat — confirm orig is untouched.
