BONUS Ternary & Nullish
▼
  1. Write a ternary: const mood = age >= 18 ? 'adult' : 'minor'; — then log it (use an age value in your script).
  2. Declare: let username = null; const display = username ?? 'Guest'; — log display.
  3. Now try: const display2 = username || 'Guest'; — does it behave the same?
  4. In a comment, explain what ?? does that || doesn’t.
Reference solution output

?? vs ||: Nullish coalescing (??) only falls back for null or undefined. || treats any falsy value (0, '', false, etc.) as “missing.”
